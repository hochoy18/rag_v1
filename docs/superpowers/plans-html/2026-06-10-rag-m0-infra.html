Goal

把 v1-scope 决策 #6（部署：Docker Compose 4 service）落成可执行基础设施：

  • 一个 docker-compose.yml 同时拉起 4 个容器（Postgres 16-alpine, OpenSearch 2.19, TEI bge-m3, Langfuse），同一 bridge 网络，service name DNS 互通
  • 一个 .env.example 覆盖全部 4 个服务的环境变量模板
  • 一个 app/config.py（Pydantic Settings）聚合全部环境变量加载
  • 一个 infra/init.sql 初始化 SQL（创建 rag DB + 基础角色）
  • 一个项目根 README.md 含启动步骤

    • 不包含（其他 M 负责）：M1 Alembic schema 迁移、M2 auth 鉴权实现、M3 LLM 工厂、M4–M6 ingest 管线、M7 graph 编排、M8 API 路由、M9 UI、M10–M12 观测/评估/hardening

DO $$ BEGIN
  IF NOT EXISTS (SELECT FROM pg_roles WHERE rolname = 'rag_app') THEN
    CREATE ROLE rag_app WITH LOGIN PASSWORD 'rag_app_password';
  END IF;
END $$;

GRANT ALL PRIVILEGES ON DATABASE rag TO rag_app;
