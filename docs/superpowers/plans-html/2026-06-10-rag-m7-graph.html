Goal

把 v1-scope §3.3 Query 数据流落成可执行 LangGraph 工作流：

  • TypedDict RagState 定义 8 节点共享状态（query / history / rewrites / chunks / answer / sources / intent / error / thread_id / user_id / trace_id）—— r2-2026-06-12：M8 r2 新-1 揭示 M8 响应字段名与 M10 trace_id 命名断裂；M7 在 state 暴露 trace_id（graph.ainvoke 返回 state 含 trace_id），M8 透传到 HTTP 响应、M10 透传到 Langfuse 上下文字段，三者统一
  • 8 节点函数：load_memory → classify → query_rewrite → retrieve → rerank → answer → save_memory_rag / save_memory_chitchat（r2-2026-06-12 新-2 拆双节点） + chitchat 分支的 answer_chitchat 节点 = 9 个节点函数（r1 标的 8 节点 = 7 主 + answer_chitchat 在 r2 拆 save_memory 后变 9）
  • 条件边：classify 节点判 intent ∈ {retrieve, chitchat} 决定后续走哪条分支；answer → save_memory 用常量边（V1 不需要路由）
  • AsyncPostgresSaver 注入到 compile()，thread_id 与 chat_sessions.thread_id 关联
  • SQLite 降级路径（PG 不可达 / setup() 权限不足时 fallback）
  • graph 调用方（M8）必须用 asyncio.wait_for(graph.ainvoke(...), timeout=30) 包装 invoke 防止 LLM 慢调用拖垮 API
  • 所有节点函数用 safe_node 装饰器包装，节点异常只设 state.error 不终止 graph
  • 端到端：graph.invoke(query, thread_id) 跑通，Postgres checkpoint tables 落数据，跨 invoke thread 复用历史

    • 不包含（其他 M 负责）：M4-M6 ingest 写 chunk 到 OpenSearch（M7 只读）、M8 FastAPI 包装（含 asyncio.wait_for 超时）、M9 Gradio 集成、M10 业务级 Langfuse trace（M3 工厂已注入了 LLM callback，M10 加节点级 trace）

    import logging
from app.config import settings
from langgraph.checkpoint.base import BaseCheckpointSaver

log = logging.getLogger(__name__)

async def make_checkpointer() -> BaseCheckpointSaver:
    """P0-3: 返回 BaseCheckpointSaver 实例，lifecycle 由调用方管理。
    P1-16: setup() 失败降级到 SQLite，不抛 PermissionError 阻塞 graph。"""
    if settings.checkpointer.dsn:
        try:
            from langgraph.checkpoint.postgres.aio import AsyncPostgresSaver
            cp = AsyncPostgresSaver.from_conn_string(
                settings.checkpointer.dsn,
                # P1-11: 显式连接池（langgraph-checkpoint-postgres 2.0.10 默认参数）
                # 默认 pool_size=5, max_overflow=10；M1 业务 DB pool_size=10+max_overflow=20
                # 总连接数 5+10+10+20 = 45 < PG max_connections=100
            )
            await cp.setup()
            log.info("checkpointer: postgres ready (pool=5+10)")
            return cp
        except (PermissionError, Exception) as e:
            log.warning("PG checkpointer setup failed (%s), falling back to SQLite", str(e))
    # SQLite 降级
    from langgraph.checkpoint.sqlite.aio import AsyncSqliteSaver
    cp = AsyncSqliteSaver.from_conn_string(settings.checkpointer.sqlite_path)
    await cp.setup()
    log.info("checkpointer: sqlite fallback at %s", settings.checkpointer.sqlite_path)
    return cp

import re

def _extract_json(text: str) -> str:
    """P1-5: 从 LLM 输出中提取 JSON 块（兼容 markdown 包裹 / 裸露 JSON / 字段名 scores 或 score）"""
    match = re.search(r"```(?:json)?\s*(\{.*?\})\s*```", text, re.DOTALL)
    if match:
        return match.group(1)
    match = re.search(r"\{.*?\}", text, re.DOTALL)
    if match:
        return match.group(0)
    return ""

def parse_json_scores(raw: str, n: int) -> list[float]:
    input_str = _extract_json(raw)
    if not input_str:
        return [5.0] * n
    try:
        data = json.loads(input_str)
        scores = data.get("scores", data.get("score", []))
        if not isinstance(scores, list) or len(scores) != n:
            return [5.0] * n
        return [max(0.0, min(10.0, float(s))) for s in scores]
    except (json.JSONDecodeError, ValueError, TypeError):
        return [5.0] * n

                }
        return wrapper
    return decorator
