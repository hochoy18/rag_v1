## Goal

把 v1-scope 决策 #5（鉴权：随机 token + SHA-256 哈希 + DB 查表，不是 JWT）、#6（密码：argon2id 哈希）、#7（越权 session_id → 404，不暴露存在性）落成可执行鉴权层：

1. `POST /api/auth/register` — 注册用户（密码强度校验 + 用户名归一 + argon2id 哈希，返回用户基本信息）
2. `POST /api/auth/login` — 登录（账号锁检查 + 密码校验，签发 32 字节随机 token，SHA-256 哈希后存 DB，**软吊销该 user 旧 session** → 单设备策略，返回 token）
3. `POST /api/auth/logout` — 登出（按 Authorization header 查 session，软吊销 → 204）
4. `GET /api/auth/me` — 鉴权 smoke endpoint（M3+ protected endpoint 样板；返回当前 user）
5. `FastAPI Depends(get_current_user)` — 路由级注入，从 `Authorization: Bearer *** 查会话、校验过期时间、必要时 `BackgroundTasks` 触发 `extend_session`
6. 限速：register 5 次/分钟/IP，login 10 次/分钟/IP（slowapi）
7. 账号锁：连续 5 次密码错误 → 锁定 15 分钟（M1 字段 `users.failed_login_attempts` / `users.locked_until`）

**不包含**（其他 M 负责）：M1 schema 表定义（已有 `users` / `auth_sessions` 表 + `is_revoked` / `email` / `failed_login_attempts` / `locked_until` / `ip_address` / `user_agent` 字段）、M3–M12 业务路由的 auth 集成（各 M 通过 `Depends(get_current_user)` 接入）、OAuth2 第三方登录、密码找回/重置流程、admin 角色管理、JWT 兼容层（V1 决策 #5 已否决 JWT）、`Dockerfile`（M8/M12 阶段补）

**模块组织原则**：
- `app/auth/` 是纯业务逻辑，不感知 HTTP 细节（无 FastAPI request 依赖）
- `app/api/auth.py` 是 HTTP 适配层，调用 `auth.service`，挂 FastAPI 限速中间件
- `app/auth/deps.py` 是 FastAPI 注入层，依赖 `auth.service` 做 token 验证
- `tests/` 镜像 `app/` 结构，单元测试全 mock DB 层

### M2 与其他 M 的契约边界

| 调用方 | 用的接口 | 备注 |
|--------|---------|------|
| M3–M12 路由 | `Depends(get_current_user)` | FastAPI 注入，返回 `User` ORM 对象；**签名**：`async def get_current_user(authorization: str = Depends(HTTPBearer())) -> User` |
| M3–M12 路由 | `request.headers["Authorization"]` | 标准的 `Bearer <token>` 格式（Header 模式，P1-12/P1-15 明确不用 cookie） |
| M2 service | M1 models: `User`, `AuthSession` | SQLAlchemy asyncio CRUD |
| M2 service | M1 字段: `users.email` / `users.failed_login_attempts` / `users.locked_until` / `auth_sessions.is_revoked` / `auth_sessions.ip_address` / `auth_sessions.user_agent` | M1 已建（M1 schema 修订后） |
| M2 config | M0 `Settings` 顶层配置 | M2 追加 `AuthSettings` 子块；`from app.config import settings` 全 M 统一约定（P2-7） |
| M1 schema | `auth_sessions.expires_at_sliding` / `expires_at_hard` | token 双过期时间列（M1 已建） |
| M2 业务层 | `username = username.lower().strip()` 统一调用入口 | Pydantic RegisterRequest/LoginRequest 归一化后传入 service（解决 M1 P1-19 大小写问题） |

- [ ] argon2id 密码哈希：time_cost=3, memory_cost=65536, parallelism=1（OWASP 2024）
